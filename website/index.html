Deploy in 30 Seconds. Pay Per Request.

# 1. Install
pip install efficient-ai

# 2. Start your own inference node
$ efficient serve --wallet 0xYOUR_WALLET --port 8000

# 3. Connect any OpenAI-compatible client
import openai
client = openai.OpenAI(
    base_url="http://localhost:8000/v1",
    api_key="not-needed"  # uses x402 payments instead
)

# 4. Make requests. Proxy routes locally first.
response = client.chat.completions.create(
    model="auto",
    messages=[{"role": "user", "content": "Process this data"}]
)
# => routed to local engine, $0.0001, 0.5ms
# => complex tasks auto-escalate to Ollama or cloud

Works with any OpenAI-compatible client. No framework lock-in.
