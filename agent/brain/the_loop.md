# The Business Loop

The core operating cycle of our AI company.

## Steps

1. **Load Context** — Read brain files for strategy, memory files for state
2. **Assess Situation** — Check revenue, leads, opportunities, content backlog
3. **Decide Action** — Use decision waterfall to pick highest-impact action
4. **Execute** — Run the chosen action via Efficient AI inference
5. **Log Results** — Update memory files with outcomes
6. **Record Telemetry** — Track inference cost, provider, savings
7. **Repeat** — Next cycle picks up where this one left off

## Key Rules

- Never skip pipeline scanning (no pipeline = no revenue)
- Always log results to memory (can't improve what we don't track)
- Keep inference costs low — use Efficient AI routing
- If an action fails, log the lesson and retry next cycle
- Content must be technically specific, not generic fluff
- Reddit comments must be helpful, not promotional

## Cycle Frequency

- Every 30 minutes during active hours
- Can be manually triggered via monitor dashboard
- Each cycle picks one action and executes it fully
