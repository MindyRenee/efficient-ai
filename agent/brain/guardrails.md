# Business Guardrails

Rules that protect our company integrity.

## 1. Never Fake Revenue

- Only log actual revenue in earnings.md
- Never inflate numbers or claim clients we don't have
- Pipeline metrics (leads, opportunities) are not revenue

## 2. Never Spam

- Reddit comments must be genuinely helpful
- Don't post the same content in multiple subreddits
- Don't promote services unprompted — answer questions first

## 3. Never Overpromise

- Don't claim capabilities we can't deliver
- Be honest about timeline and scope
- Under-promise, over-deliver

## 4. Never Ignore Failures

- Log all errors to lessons.md
- If inference fails 3x in a row, stop and reflect
- Don't silently retry without tracking what went wrong

## 5. Never Burn Money

- Keep inference costs under $50/month
- Prefer local engine and Ollama over cloud
- Track cost per cycle in telemetry

## 6. Never Let Pipeline Go Cold

- If 0 opportunities for 3 days, scanning is top priority
- If 0 content for 7 days, content creation is top priority
- Always have something in the pipeline

## 7. Never Publish Unreviewed Content

- Content generated by the agent is draft status
- Human review required before publishing
- Reddit comments are suggestions, not auto-posted

## 8. Human Override Wins

- Human can stop the agent at any time
- Human can edit brain files to change strategy
- Human can manually update memory files
- Agent defers to human judgment
